i have already make format list in my plugin combobox to export my analysis data, the format list given bellow

GeoTIFF (.tif/.tiff) : Industry standard raster format with embedded geospatial metadata and coordinate reference system
JPEG2000 (.jp2) : Advanced compression format supporting both lossless and lossy compression
PNG (.png) : Lossless compression format with transparency support, ideal for classification results
JPEG (.jpg) : Widely compatible compressed format suitable for visual inspection and quick result sharing
BMP (.bmp) : Uncompressed bitmap format with simple structure, resulting in large file sizes but maximum compatibility
TIFF (.tiff) : Flexible raster format without geospatial tags, supporting multiple layers and high-quality storage
PDF (.pdf) : Portable document format preserving layout and quality, ideal for report generation and documentation
Shapefile (.shp) : ESRI's widely supported format for points, lines, polygons with attribute data
GeoJSON (.geojson) : Lightweight, web-friendly JSON-based format for geospatial features
KML/KMZ (.kml/.kmz) : Google Earth format for geographic visualization and sharing
GPKG (.gpkg) : Modern SQLite-based format supporting both raster and vector data
DXF (.dxf) : AutoCAD format for CAD and GIS data exchange
